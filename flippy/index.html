How does it work?

Flippy uses CSS3 3D transforms & transitions, all made manageable with Compass.

It falls back to an opacity fade & CSS visibility in unsupported browsers using Modernizr.

Flippy is inspired by the "shout-outs" animation at CSS Tricks.
